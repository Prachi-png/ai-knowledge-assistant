# 🤖 AI Knowledge Assistant (RAG Chatbot)

An AI-powered chatbot built using Streamlit, LangChain, ChromaDB, and Google Gemini API.  
Supports PDF-based question answering using Retrieval-Augmented Generation (RAG).

---

## 🌐 Live Demo

👉 https://ai-knowledge-assistant-chatbot.streamlit.app/

---

## 🚀 Features

- 💬 Chat-based interface
- 📄 Upload PDF and ask questions
- 🧠 RAG (Retrieval-Augmented Generation)
- ⚡ Fast with caching
- 🔍 Context-aware answers

---

## 🛠️ Tech Stack

- Python
- Streamlit
- LangChain
- ChromaDB
- HuggingFace Embeddings
- Google Gemini API
